Код програми реалізації алгоритму Евкліда знаходження найбільшого спільного дільника: 

   function nod( a, b: longint): longint; 
  begin
   while (a <> 0) and (b <> 0) do
     if a >= b then 
       a:= a mod b 
     else 
       b:= b mod a;
   nod:= a + b;
  end;
  

    Алгоритм Евкліда для цілих чисел:
	
 Нехай a і b - цілі числа, не рівні одночасно нулю
 a> b> r1> r2> r3> r4> ...> rn , то:
a = bq0 + r1,
b = r1q1 + r2,
r1 = r2q2 + r3,
...
rk-2 = rk-1qk-1 + rk,
...
rn-2 = rn-1qn-1 + rn,
rn-1 = rnqn

Доведення:

ОДЗ (r, 0) = r для будь-якого не нульового r.
